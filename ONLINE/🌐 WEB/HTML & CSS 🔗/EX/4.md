Exercícios de Aplicação Página 1

1. Cria um ficheiro HTML com a estrutura mínima obrigatória para uma página web. Inclui a declaração do tipo de
documento, o elemento <html> com os seus elementos filhos <head> e <body>, e dentro do <head> adiciona um
título apropriado para a página.
a. Adiciona uma folha de estilos ao ficheiro HTML do exercício 1 e aplica os seguintes estilos:
i. Define a cor de fundo do <body> como lightblue.
ii. Define a cor do título (<h1>) usando um valor hexadecimal.
iii. Define a font-family de todo o corpo da página para "Arial", sans-serif.
2. Adiciona ao teu ficheiro HTML três imagens diferentes usando a tag <img>.
Cada imagem deve ter um alt descritivo, e pelo menos uma delas deve usar um atributo width ou height para
controlar o seu tamanho. Usa imagens de URLs externas ou locais.
a. Com base no ficheiro HTML do exercício 2, faz o seguinte:
i. Aplica uma borda preta de 1px sólida a todas as imagens com o atributo alt.
ii. Define a cor de fundo (background-color) dos parágrafos que vêm imediatamente antes das
imagens para rgba(255, 255, 0, 0.2).
iii. Define o tamanho da fonte (font-size) dos parágrafos para 14px.
3. Cria uma secção com um título, dois parágrafos com quebras de linha onde fizer sentido, e um bloco de código
usando a tag <pre>. Inclui também um poema ou texto com estilo livre.
a. A partir do conteúdo do exercício 3:
i. Usa uma classe para aplicar font-family: "Courier New", monospace ao poema.
ii. Define a cor do <h2> usando valores rgb.
iii. Remove o sublinhado dos <h3> com text-decoration: none.
4. Adicionar aos elementos HTML criados os atributos globais class e id que julgar necessários.
a. Baseando-te no exercício 4:
i. Usa CSS para aplicar color: green aos parágrafos com a classe intro. Não uses style inline.
ii. Altera o tamanho da fonte (font-size) do parágrafo com um id à tua escolha para 18px.
iii. Aplica um background-color a qualquer elemento que tenha simultaneamente um id e uma
class, usando um seletor CSS combinado.
5. Cria uma frase que utilize diferentes elementos semânticos do HTML para dar significado ao texto. Utiliza também
um <span> com uma classe.
a. Com o ficheiro HTML do exercício 5:
i. Aplica ao <span> com a classe destaque a cor red e um tamanho de letra de 20px.
ii. Define um background-color com rgba para o conteúdo dentro da tag <mark>.
iii. Remove o sublinhado de todos os elementos <cite>

6. Cria uma página HTML completa com estrutura semântica moderna: inclui um <header>, <main> com <section> e
<article>, uma <aside> e um <footer>. Utiliza também classes e atributos relevantes para organização e estilo.

a. No exercício 6:
i. Aplica uma cor de fundo (background-color) ao <header> e ao <footer>.
ii. Define uma cor de texto (color) diferente para <main>, <section> e <article>.
iii. Aplica uma font-family ao <aside> apenas quando ele estiver dentro de um <main> (usa seletores combinatórios)
