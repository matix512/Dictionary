Criar o código base: no IDE escrever ! e dar enter e : 

```html
<!DOCTYPE html>

<html lang="en"> - Idioma

<head>

    <meta charset="UTF-8">

    <meta name="viewport" content="width=device-width, initial-scale=1.0">

    <title>Document</title>

</head>

<body>

</body>

</html>
```

Dentro do HTML existem duas áreas : Head e Body

Head - Configurações
Body - Conteúdo
